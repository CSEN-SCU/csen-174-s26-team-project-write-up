# Reader-centered clarity

Readers track **actors and actions**. When a sentence contains many ideas at once, they may understand each word yet still lose the thread.

A practical test: can someone answer “who did what, and why does it matter here?” after a single read? If not, the issue is often **information order**, not vocabulary level.

# Optional micro-edits

A micro-edit is a **small surgical change**: swap clause order, add a two-word bridge, or split at a natural breath. It should not replace the author’s voice; it demonstrates one possible tightening.
